\# AI\_RULES (Read before coding)



\## Context

This is a Unity ADV prototype.

Keep everything simple, readable, and shippable.



\## Engine / Tools

\- Unity: 6000.3.0f1

\- Use uGUI + TextMeshPro by default

\- Git LFS and Smart Merge are enabled



\## Constraints

\- Avoid over-engineering

\- No complex architecture early

\- One feature at a time



\## Coding Rules

\- Scripts go under `Assets/\_Project/Scripts`

\- Use namespace: AdvPrototype

\- Prefer clarity over cleverness



\## Output Requirements

When generating code:

1\. Show file paths

2\. Provide full code

3\. Mention required Unity Editor steps

4\. Keep changes minimal



\## First Milestone

\- Title screen

\- Dialogue box (TextMeshPro)

\- Advance text on click / key

\- Dialogue data via ScriptableObject or JSON
