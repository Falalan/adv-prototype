\# adv-prototype



Unity ADV prototype project.



\## Goal

Build a small shippable ADV prototype (itch.io / Steam later).

Focus: simple UI, readable text, fast iteration.



\## Engine

\- Unity: 6000.3.0f1 (Unity 6)

\- Git LFS: enabled

\- Smart Merge (UnityYAMLMerge): enabled



\## Rules

\- First-party assets: `Assets/\_Project/`

\- Third-party assets: `Assets/ThirdParty/`

\- Prefer small commits that compile
